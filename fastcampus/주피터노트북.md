# 주피터 노트북 사용법

## 주피터 노트북

→ 코드나 주석 입력 가능

녹색: 입력

파랑: 명령 (esc)

### 단축키

h: help (단축키)

a: 위에 추가

b: 아래 추가

dd: 삭제

shift+enter: 실행 후 다음 shell 이동 (없을 경우 새로 생성)

option + enter: 실행 후 새로운 shell 생성

control+enter: 실행

### type

코드: In [] -> input[횟수]

마크다운: Markdown

### type 변경

Cell -> Cell Type -> Code/Markdown
